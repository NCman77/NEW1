# 測試資料夾

此資料夾包含前後端分離改造的測試基準檔案。

## 檔案說明

### test-data.json
包含測試案例定義和預期輸出格式：
- 7 個核心測試案例
- 涵蓋所有遊戲類型（大樂透、威力彩、今彩539）
- 涵蓋所有主要學派（均衡、AI、統計）
- 涵蓋所有模式（嚴選、隨機、包牌）

### validation.js
自動化驗證腳本：
- 驗證輸出格式正確性
- 檢查號碼範圍和數量
- 檢查無重複號碼
- 提供詳細的錯誤報告

## 使用方式

### 執行測試（需要先安裝 Node.js）

```bash
# 在專案根目錄執行
node test/validation.js
```

**Phase 1（當前）：格式驗證**
- ✅ 驗證測試資料格式正確

**Phase 2-3：後端整合**

## 驗證標準

### 必須通過的檢查
- ✅ 號碼數量正確
- ✅ 號碼範圍正確
- ✅ 無重複號碼
- ✅ 包含必要的 metadata
- ✅ 資料結構完整

### 效能標準（Phase 4）
- ⏱️ API 回應時間 < 2 秒
- ⏱️ 包牌模式 < 5 秒
- ⏱️ 無記憶體洩漏

## 注意事項

⚠️ **重要**：此測試僅驗證輸出格式，不驗證號碼選擇邏輯的正確性。

🔒 **核心算法保護**：測試過程不會修改任何算法邏輯。
